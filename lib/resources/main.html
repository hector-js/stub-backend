@hectorjs/stub-backend

NPM library, Github

Resources

 

        
How to

        

            
Description

            
The aim of this project is to mock backend services building different responses for a given
                authentication, cookie, request among others.

            
Make the life easier for a nice development

        

        

            
Install

            
npm install -g @hectorjs/stub-backend

        

        

            
CLI

            
Verify your hjs command is installed correctly.

            

                hjs --version
            

            

                hjs test
            

            
You should see a health test generated by default.


            

                
Commands

                
    
                    
  • 
                        
    
                            
                            
    It will display posible options available in the library.
    
                        
    
                    
    • 
                    
  • 
                        
    
                            
                            
    It creates a new project with the name provided in the brakets.
    
                        
    
                    
    • 
                    
  • 
                        
    
                            
                            
    It will generate a method template resource with a test. For example ,
                                hjs g get /customers). More info: hjs g --help.
    
                            
    You can generate different methods like get, head, post, delete among
                                others.
    
                            
    For each method, it creates a test with dummy data. It will reduce the time
                                to test the mock. Give it a try.
    
                        
    
                    
    • 
                    
  • 
                        
    
                            
                            
    Execute the tests which have been created by each method generate command
                                (hjs g get ...).
    
                            
    Note: you can run npm test as well.
    
                        
    
                    
    • 
                    
  • 
                        
    
                            
                            
    If you want to run the mock hjs start.
    
                            
    Note: you can run npm start as well.
